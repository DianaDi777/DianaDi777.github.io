SLIDESHOW

Full screen slideshow for header

FUNCTIONALITY

JS will:
(a) listen for a user click on arrow-key to go to the next slide.
(b) autoplay slides when the page is loaded. It stops the autoplay when a user clicks the arrow-key. It starts autoplay if a user is not clicking arrow-key anymore.
(c) create an info bar that will show how many slides there are and which one is currently playing.
(d) create a ticker at the top of the slideshow to present how long in the current slide will appear on the screen.
CSS will:
(a) zoom in on a picture when it appears on the screen
(b) provide some animation for a smooth color transition when a user hovers on the arrow button and other transitions.
* This slide show does not display on medium and small screens
