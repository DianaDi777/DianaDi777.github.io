Navigation bar

Responsive main navigation menu

FUNCTIONALITY

Depending on the size of the viewport, the navbar will either be displayed in one of two ways.
( I ) One way is for all the links to be on a single row, where each link is evenly spaced.
( II ) A second way is for the links to be tied to a hamburger menu, where the links will appear on top of each other in one column.
This will use a combination of JS and CSS.
The JS will check for the media query and add/remove the applicable CSS classes automatically.
CSS changes the styling of the links in the default state, on hover, and when clicked.
All of this functionality will be available to the user by adding one specific class from our framework to their HTML.

Download our framework filles and link it into your file.
