Uneven Columns

To make uneven columns, add a class "uneven" to the parent element. Then add a class "col-#of12", where # is a number between 1 and 12. The width of the columns will be automatically calculated based on the relative sizes to the other columns.

Uneven Column 1

Uneven Column 2

          
            <div class = "uneven">
              <div class = "col-6of12">
                  <p> Uneven Column 1 </p>
              </div>
              <div class = "col-3of12">
                  <p> Uneven Column 2 </p>
              </div>
            </div>

Grid Example: 4x3 grid

One way to make a 4x3 grid is to make the parent grid element the number of colums by using the class "col-#", where # is the number of overall columns desired. In this example "col-4" is used because 4 columns are desired.

To make the number of rows, add an embedded column class of 1 column in each grid child element. This column class will control the number of rows by adding child items to that grid child element. In this example, "col-1" is used in each grid child element. Then the children of that grid child element control how many rows that column will have.
