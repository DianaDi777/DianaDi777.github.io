Table Color

To set the font color of any table row or element, add the "color-#" class, where # is replace with any desired color. The color can be in english or in hexadecimal. This class can be added to any tr, th, or td element.

Table Starter

To make a table, add any of the following classes to the table element:

  • "centerGroup" - centers the table horizontally from the parent (optional)
  • "border-outline" - makes a border around the table (optional)
  • "border-row" - makes a border around the rows (optional)
  • "border-grid" - makes a border around the individual grid cells (optional)
  • "responsive-table" - makes the table responsive (optional)
  • "stripes" - puts a gray background color on alternating rows (optional)
