-- max width--

Class "MaxW-narrow"

Element has max-width equals to 20 em. It's not going to be bigger than that. You can use it then you want to keep your element small.

Class "MaxW"

Element has max-width equals to 30 em. It's not going to be bigger than that. You can use it then you want to keep your element mid size.

Class "MaxW-wide"

Element has max-width equals to 34 em. It's not going to be bigger than that. You can use it then you want to keep your element wide.
