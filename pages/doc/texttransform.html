--Transformation styling--

If you need capitalize your text use class 'capit'

If you need uppercase your text use class 'up'

If you need lowercase your text use class 'low'

If you need to indicate that the text is a short fragment of computer code like this: querySellector() use class 'code'.

If you need to indicate that the text is a citation like this: All that glitters is not gold use class 'cite'.

--Bold and Italic--

If you need italicize style use class 'ital'

If you need bold style use class 'bold'

--text decoration styling--

If you need underline your text like this: Brevity is the soul of wit use class 'line'.

If you need underline your text like this: If music be the food of love, play on. use class 'underline'.
