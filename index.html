INSTRUCTIONS HOW TO DOWNLOAD AND USE

Our framework is all about helping local produce vendors with our E-commerce Website Framework! It is easy to use and free to download! Just follow the instructions and your business will experience the benefits of the world wide web in no time!

  1. Download zip file by clicking the button above.
  2. In the HTML file, link to the downloaded CSS file using the <link href = "[file_location]"> element in the head section.
  3. In the HTML file, link to the downloaded JS file using the <script src = "[file_location]"> element in the bottom of the body section.
  4. Assign the appropriate classes to the tags you want to apply to your website!
